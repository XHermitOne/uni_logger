AUTOMATION WRAPPER
------------------

Graybox OPC HDA Automation Wrapper is a DLL-module, in which all of the needed
OLE-objects are implemented.

After registering this module, you will be able to use any OPC Historical
Data Access Server with almost any OLE enabled programming language
(Visual Basic, VBA, etc).


USAGE
-----

Do the following steps to install the Wrapper.
- copy gbhda_aw.dll to Windows\System32 folder;
- register this module - enter regsvr32 gbhda_aw.dll in the command line.

To remove Graybox OPC HDA Automation Wrapper from your system's registry enter
regsvr32 gbhda_aw.dll -u.
